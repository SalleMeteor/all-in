Betit (old All-in)
==================

App description 
===============
Website with inital login where you can bet for football matches and simulate them. Moreover, depends on your bet, you can win (or lose!).

It uses 
=======
- standard-app-packages
- bootstrap
- accounts-base
- accounts-password
- accounts-ui
- iron-router
- nprogress
- blaze-layout
- bootstrap-3

Deployed URL
===========
http://betit.meteor.com
